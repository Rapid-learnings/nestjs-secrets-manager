# About The App

NestJS template with AWS Secret Manager integration, to fetch secrets synchronously and load it into built-in config service.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# watch mode
$ npm run start:dev

# staging mode
$ npm run start:staging

# production mode
$ npm run start:prod
```
